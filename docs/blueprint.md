# **App Name**: Sinal Intel Hub

## Core Features:

- Automated Report Generation: Automatically generate performance reports for each car brand from Meta Ads data. The LLM will use a tool to analyze Meta Ads data and generate insightful summaries for each brand.
- Data Source Connection: Connect to Google Sheets to fetch the data and allow periodic refresh.
- Interactive Dashboard: Provide a dashboard with clear visualizations of the results for each brand.
- Key Performance Indicators (KPIs): Display essential KPIs like total leads, CPL, and other relevant metrics.
- Brand Performance Comparison: Allow users to compare the performance of different brands side-by-side.
- Customizable Date Ranges: Enable users to select custom date ranges for report generation.

## Style Guidelines:

- Primary color: Deep blue (#0054A6) for Sinal's brand recognition.
- Background color: Dark navy (#0B0F19) for a modern, sleek appearance. This works well in a dark color scheme.
- Accent color: Electric blue (#00A654), an analogous hue to the primary, but significantly brighter and more saturated.
- Body and headline font: 'Poppins', a sans-serif, for a clean and modern user interface.
- Use Font Awesome icons for visual appeal and consistency.
- Utilize a grid layout for organizing KPIs and charts effectively.
- Subtle animations for loading states and data updates to enhance UX.
